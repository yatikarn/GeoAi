# GeoAI

ผลงานฝึกอบรม GeoAI & Geo-Informatics — GISTDA satellite price list extraction, shapefile/GeoJSON conversion, and Google Earth Engine (Sentinel-2) analysis for Thailand.

## Live demos

Once GitHub Pages is enabled for this repo (Settings → Pages → Source: `main` branch, `/ (root)`), these will be viewable directly in a browser:

- [Sentinel-2 median composite over Thailand](Sentinel2.html) — cloud-light (≤10%) Sentinel-2 SR composite, 2026-01-01 to present, on a Google Satellite basemap. Generated by [`Sentinel2.py`](Sentinel2.py).
- [GISTDA satellite price comparison chart](Gistda_Price_List_comparison.html) — interactive Archive vs Tasking price bar chart. Generated by [`gistda_price_extractor.py`](gistda_price_extractor.py).
- [Thailand boundary map](Example/Thailand.html) — Thailand shapefile converted to GeoJSON and rendered as an interactive Leaflet map. Generated by [`shp_to_geojson.py`](shp_to_geojson.py) + [`geojson_to_html.py`](geojson_to_html.py).

## Contents

| File / folder | Description |
| --- | --- |
| `gistda_price_extractor.py` | Downloads the GISTDA satellite price list PDF and extracts a structured price table using `pdfplumber` (real table cells, not guessed text tokens) |
| `Gistda_Price_List_structured.{csv,xlsx,html}` | Structured price table output (all satellites/SAR modes, per-page section labels) |
| `Gistda_Price_List_comparison.html` | Plotly grouped bar chart, Archive vs Tasking price by satellite |
| `Gistda_Price_List.{csv,xlsx,html,pdf}` | Raw page-text dump + original source PDF |
| `shp_to_geojson.py` | Converts a shapefile to WGS84 GeoJSON (reprojects from the shapefile's native CRS) |
| `geojson_to_html.py` | Renders a GeoJSON file as a standalone interactive Leaflet HTML map with a Google Maps basemap |
| `Example/` | Thailand shapefile + POI data (from GEE4NRE course materials) and its converted GeoJSON/HTML outputs |
| `Sentinel2.py` / `Sentinel2.html` | Earth Engine script producing a cloud-light Sentinel-2 median composite over Thailand |

## Setup

```
python -m venv .venv
.venv\Scripts\pip install -r requirements.txt   # see individual scripts for the packages each one needs
earthengine authenticate
earthengine set_project <your-gee-project-id>
```
